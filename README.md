# Spotcode

PWA feito nas aulas do OneBitCode. Foi baseado na interface do Spotify, tem o front-end feito em React e o back-end feito em Ruby on Rails.Desenvolvido com propósitos de prática e estudo, somente.

 - Ruby on rails
 - Jbuilder
 - Deviser
 - sqlite3
 
 - React
 - React Router
 - Styled Components
